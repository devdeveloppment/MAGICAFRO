# MagicAfro

Une boutique de produits capillaires et cosmétiques haut de gamme, dédiée à la beauté naturelle.

## Caractéristiques
- Catalogue de produits par catégories
- Système de panier et commande
- Blog intégré
- Design premium White & Gold
- Intégration de paiements (CinetPay/Stripe)

## Installation
1. Cloner le repository
2. Créer un environnement virtuel
3. Installer les dépendances : `pip install -r requirements.txt`
4. Configurer le fichier `.env`
5. Lancer les migrations : `python manage.py migrate`
6. Lancer le serveur : `python manage.py runserver`
